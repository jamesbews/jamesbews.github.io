CrunchBite

Current, Personal Project

Ionic Angular Node.js MongoDB AWS GitLab CI/CD Firebase

The goal of this project was to improve my full stack technical skills by building a cross-platform mobile and web application. I choose meal planning because it's a subject which I don't believe any app on the market does very well. I often find myself at the grocery store wondering what to buy just to end cooking the meals every week. I wanted to shake it up a bit.

The app consists of 3 sections; searching for recipes, creating a meal plan and creating a grocery list. The user can favourite recipes and use those to create a meal plan for the week. From the meal plan, every ingredient from each recipe is aggregated into a simple grocery list that can be used to buy groceries.

The project was rewarding and challenging. I was able to learn new skills and technologies that are both interesting to me and valuable to my progression as a Full Stack Developer. Being able to build each component from the database, to the server, to the frontend; exposed me to technologies I am not able to interact with as regularly as I would like. It also introduced me to hands-on experience with AWS deployments and using GitLab to automate that process, skills that I can bring to the table in my professional career.

I am currently using the app to plan my weekly meals and buy groceries! Recently, most of the development has been UI/UX improvements and bug fixes.

Notable Features:

  • Cross-Platform deployment with single code base
  • Firebase JWT Authentication
  • GitLab Automated Deployments
  • Deployed MongoDB Instance
